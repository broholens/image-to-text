# image-to-text
通过百度文字识别API，将图片转为可复制的文字。目前支持手机号和邮箱地址的转换。

邮箱地址的转换支持如下:
- 126.com
- qq.com
- 163.com
- gmail.com
- sina.com
- hotmail.com
- outlook.com
